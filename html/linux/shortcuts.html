Red Hat Linux Shortcut Keys

General Shortcuts

Action Shortcut Use Case
Open Terminal Ctrl + Alt + T Quickly access the command line interface.
Open a new terminal window Ctrl + Shift + N Open an additional terminal window without using the mouse.
Open a new tab in the terminal Ctrl + Shift + T Organize terminal sessions in tabs within a single window.
Copy selected text Ctrl + C Copying text to the clipboard.
Paste copied text Ctrl + V Pasting text from the clipboard.
Cut selected text Ctrl + X Removing and copying text to the clipboard.
Undo the last action Ctrl + Z Reverting the previous command or text input.
Select all text Ctrl + A Quickly selecting all text for copying or cutting.
Open the find dialog Ctrl + F Searching for text within a document or webpage.
Save the current document or file Ctrl + S Save changes to avoid data loss.

Terminal Shortcuts

Action Shortcut Use Case
Clear terminal screen Ctrl + L Clearing clutter to focus on new commands.
Cancel current command Ctrl + C Interrupt a running process.
Logout from current terminal session Ctrl + D End a terminal session or close a shell.
Search command history Ctrl + R Quickly find and reuse previous commands.
Delete line before cursor position Ctrl + U Quickly remove the current line of text.
Delete line after cursor position Ctrl + K Quickly remove text from the cursor to the end of the line.
Delete word before cursor Ctrl + W Quickly remove the previous word.
Move cursor to end of the line Ctrl + E Quickly navigate to the end of the command line.
Move cursor to beginning of the line Ctrl + A Quickly navigate to the start of the command line.
Move cursor forward one word Alt + F Quickly navigate through the command line.
Move cursor backward one word Alt + B Quickly navigate through the command line.

Window Management Shortcuts

Action Shortcut Use Case
Switch between open applications Alt + Tab Quickly switch between running applications.
Close the current window Alt + F4 Quickly close the active application.
Open the activities overview Super (Windows) Key Access the overview to manage windows and applications.
Tile window to the left or right half of the screen Super + Left/Right Arrow Snap windows for side-by-side multitasking.
Maximize the current window Super + Up Arrow Maximize the application for full-screen view.
Restore or minimize the current window Super + Down Arrow Minimize or restore the application window.

Text Editing Shortcuts

Action Shortcut Use Case
Move cursor left or right by one word Ctrl + Left/Right Arrow Navigate text quickly by words.
Select text by word Ctrl + Shift + Left/Right Arrow Quickly select words for copying or cutting.
Move cursor to beginning of the line Home Quickly navigate to the start of the line.
Move cursor to end of the line End Quickly navigate to the end of the line.
Move cursor to beginning of the document Ctrl + Home Quickly navigate to the start of the document.
Move cursor to end of the document Ctrl + End Quickly navigate to the end of the document.
Delete the word before the cursor Ctrl + Backspace Quickly remove the previous word.
Delete the word after the cursor Ctrl + Delete Quickly remove the next word.
